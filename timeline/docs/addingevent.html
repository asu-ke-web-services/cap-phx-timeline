TImeline File Structure

index.html - contains the HTML content for SPLASH page
about.html - contains the HTML content that shows up after clicking on "about" tab in the timeline
help.html - contains the HTML content that shows up after clicking on "help" tab in the timeline
categories/ - contains the HTML files for categories displayed in the timeline - EDIT FILES TO ADD EVENT BOXES
events/ - contains categorized event details HTML files - ADD FILES TO CATEGORY FOLDERS FOR EVENT HTML FILES (DETAILED VIEW OF EVENT)


These are to be edited only if adding categories:
timeline.html
css/ - contains all CSS style sheets for the entire project

These folders and files are not to be edited: 
images/- contains timeline specific images for styling the timeline
scripts/ - contains javascripts
fancybox/ - contains javascript that enables to position content over timeline

These are Reference material - no need to upload to server:
docs/ - has documentation for working with timeline
image-sources/ - has photoshop files for images used - Scale, Slider

Creating Event Boxes in Timeline Categories

This is the piece of code to add into your category timeline file (located in "categories/")

Our eventID convention: e%YEAR%-%SEQUENTIAL% (for example: the first event in year 1860 would be e1860-001

<div id="%%eventID%%" class="eventOuter ep%%YEAR%%a"> <div class="eventInner pLeft"> <div class="content linked" onclick="javascript:parent.$.fancybox.open({ href: 'events/%%category%%/%%eventID%%.html',type: 'iframe'}); "> <h1>%%YEAR%%-%%LASTYEARINRANGE%%</h1> <h2>IN ALL CAPS ARE HEADERS</h2> <p>Simple text is marked as a paragraph but is the same size as the header, the text is aligned to center.</p> </div> </div> </div>

Let's consider an event with the following parameters:
- in "agriculture" category
- occurs in 1860 (not a range of years)
- has a linked event details file
Code for that event would be:

<div id="e1860-001" class="eventOuter ep1860a">
<div class="eventInner pLeft">
<div class="content linked" onclick="javascript:parent.$.fancybox.open({ href: 'events/agriculture/e1860-001.html',type: 'iframe'});">
<h1>1860</h1>
<h2>IN ALL CAPS ARE HEADERS</h2>
<p>Simple text is marked as a paragraph but is the same size as the header, the text is aligned to center.</p>
</div>
</div>
</div>

Note about year: we are using the first year in range for position

Not linked event would look like this:

<div id="%%eventID%%" class="eventOuter ep%%YEAR%%a"> <div class="eventInner pLeft"> <div class="content"> <h1>%%YEAR%%-%%LASTYEARINRANGE%%</h1> <h2>IN ALL CAPS ARE HEADERS</h2> <p>Simple text is marked as a paragraph but is the same size as the header, the text is aligned to center.</p> </div> </div> </div>

Event Types:
- Simple Unlinked Centered Text or Headers
- Simple Linked Centered Text or Headers
- Event Box that includes a Thumbnail and other content
- Event Box that includes just a Thumbnail

Add the box to category file

Now that we have our box html we are going to add it to the category timeline file. In this instance we open file "agriculture.html" in "categories/" folder

Adding Event Details HTML Documents

1. Create an HTML document (from template or blank)
2. Save this document as "%%eventID%% which looks like this e1860-001 to a file in "events/" folder under the category it resides in (for example: "agriculture/"). Filename would look like: "e1860-001.html"
